overview

해양수산R&D지식정보시스템 기존 홈페이지 내에 구성 되어 있던 기술거래 메뉴 내 페이지를 [기술거래 플랫폼]으로 독자적으로 구성하였다. 플랫폼으로 구성 하면서 정체성은 잃고 싶지 않다는 고객의 니즈가 있어, 해양수산부의 포인트 컬러인 블루 계열의 컬러를 트렌드한 느낌으로 밝게 변경하여 사용하였고 기존 페이지와 달리 시원하고 확장된 느낌을 주고자 메인페이지는 원페이지 형식으로 구성하였다.

  • MY ROLE

    프론트 페이지, 관리자 페이지 전체 UI

  • DATE

    2020.07 ~ 2020.09

  • TYPE

    데스크탑 웹

  • CONCEPT

    독자적인 플랫폼으로 구성 하지만, 해양수산과학기술진흥원의 정체성을 유지하여 연결된 홈페이지임을 사용자가 인지하도록 구성
  • 블루계열 색상
    심볼마크
    정체성 유지

  • COLOR PALLETTE

    • 해양수산부의 아이덴티티를 살리기 위해 블루컬러를 주조색으로 사용하고 싶다는 고객의 니즈를 충족시킴과 동시에 단조롭고 딱딱한 이미지를
    벗어나기 위해 밝은 컬러를 사용하고, 포인트 컬러의 그라디언트 적용 등으로 트렌디한 느낌을 주었다.

    TYPOGRAPHY

    • 다양한 weight, 다국어를 지원하고, 저작권에 문제가 없는 오픈소스 글꼴 Noto Sans CJK 를 시스템 폰트로 지정.

    ICONGRAPHY

    • 라인 형태에 포인트 컬러만 적용해 심플하면서도, 다소 어려운 해양수산자원 관련 언어를 이용자에게 이해시키기 위해 최대한 직관적으로 아이콘 구성.

    GRID SYSTEM

    • 국내 데스크탑 해상도 최대 점유율 1920px을 기준으로( 출처 : Statcounter / 통계 기간 : ‘2019.10. ~ ‘2020.10. ) 작업하였으며,
    해상도 1280px의 19인치 모니터에서도 정상출력 가능하도록 1200px 센터 고정 콘테이너 내에 콘텐츠를 위치하였다.
